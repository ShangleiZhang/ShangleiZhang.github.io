Dev Workshop Landing Page

  • Sept,2016
  • Front-End Dev

Project Info

This is a landing page prototype for Dev Workshop LCC. The landing page requires a top navigation that has a login and a sign up button. As also, a logo specially designed for "Dev Workshop" was expected.In the footer area, client wants to present the contact information and newsletter sign up option.


When the user clicks on either the signup or login, it should pull up a pop up that has a form for either of these, and could be switchable if needed.

* This is an individual project.
